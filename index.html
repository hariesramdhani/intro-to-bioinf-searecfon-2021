This will be the site containing the resources whether it is homework or course materials for the Introduction to Bioinformatics with SEAMEO RECFON and Action Against Stunting in March 2021.

Homework and Materials

During the training there will be several homeworks given after each of the session, doing the homework is not mandatory but it's nice to be done so the students will still remember the topic that is being studied.

Intro to Python for Bioinformatics #1

During the session the students will be introduced to programming language and will learn some basic Python. For the homework homework the students will try to apply some basic Python on biological problems that they might encounter. There are two problems in the homework.
• Problem 1: COVID Protein
• Problem 2: DGEA Experiment Results

Intro to Python for Bioinformatics #2

During the session the students will be introduced to IF ELSE, LOOP. For the homework the students will try to apply a little bit more advanced Python on biological problems that they might encounter. There are two problems in the homework.

Intro to Python for Bioinformatics #3

During the session the students will be introduced to library and some basic data analysis in Python using pandas. For the homework the students will try to analyze the given expression data from a publicly available dataset from GEO. There are two problems in the homework.
